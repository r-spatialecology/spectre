# plot_commonness

Plot commonness between observed and optimized data

## Usage

``` r
plot_commonness(x, target)
```

## Arguments

- x:

  Results object of run_optimization_min_conf()

- target:

  Pairwise matrix of species in common.

## Value

ggplot

## Details

Plot a heatmap of commonness between observed data and optimized data.
This visual style allows for easier spatial understanding of commonness
differences to be ascertained.
